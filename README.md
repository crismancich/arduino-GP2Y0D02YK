# arduino-GP2Y0D02YK
Read the digital sensor output from the SHARP Long Distance Measuring Sensor GP2Y0D02YK

Reads the digital sensor output to the pc console via serial output

 The circuit:
 * Sensor VCC to 5V
 * Sesnor GND to GND
 * Sensor VO to digital pin 7
